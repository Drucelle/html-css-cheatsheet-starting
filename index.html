HTML <head> Cheat Sheet

Requirements Key

  1. ✅Essential - Must be included in every HTML document
  2. ⭐Recommended - Best practice to include in most projects
  3. 🔵Optional - Include only when needed for your project
Order Required Tag Name Description
1. ✅ <!DOCTYPE html> Document Type Tells the browser this is an HTML5 document. Always the first line of every HTML file.
2. ✅ <html lang="en"> Root Element Wraps all page content. The lang attribute tells the browser the page language important for screen readers and accessibility.
3. ✅ <meta charset="UTF-8"> Character Encoding Sets the character set so all letters, symbols and emojis display correctly.
4. ✅ <meta name="viewport" content=”width=device-width, initial-scale=1.0”> Viewport Makes the page responsive on all screen sizes. Essential for mobile first design.
5. ✅ <title> Page Title Sets the text that appears on the browser tab and in Google search results.
6. ⭐ <meta name="description"> Meta Description Tells Google what your page is about. Appears as the summary in search results.
7. 🔵 <meta name="author"> author Identifies who wrote the page.
8. 🔵 <link rel="icon" href="./assets/images/favicon.png"> Favicon Sets the small icon that appears on the browser tab next to your page title.
9. 🔵 <link rel="preconnect" href="https://fonts.googleapis.com"> Google Fonts Preconnect First connection to Google Fonts server. Always needed before the font link.
10. 🔵 <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> Google Fonts Gstatic Second connection to Google's font file server. Always paired with the first.
11. 🔵 <link href="https://fonts.googleapis.com/css2?family=Font+Name:wght@Weight&display=swap" rel="stylesheet"> Google Fonts Stylesheet Loads the actual font. Replace Font+Name and Weight with your chosen font.
12A. ⭐ <link rel="stylesheet" href="./styles.css"> CSS Link In Same Folder Use this when your CSS file is in the same folder as your HTML file. The ./ means "right here".
12B. ⭐ <link rel="stylesheet" href="./assets/css/styles.css"> CSS Link Subfolder Use this when your CSS file is inside a subfolder. The path shows the browser exactly where to look.
